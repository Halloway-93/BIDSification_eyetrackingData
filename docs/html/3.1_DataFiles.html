DataFiles¶

Files containing raw eye position data (in .asc, .csv or .tsv )

DataFiles.asc¶

The DataFiles.asc files are the raw data files from the Eyelink tracking system converted to an asc file.

DataFiles.csv or DataFiles.tsv¶

The DataFiles.tsv or DataFiles.csv files must contain the eye-tracking data.
They must be in the same form as the filename_eyetrack.tsv.gz file.

Warning

The data must be separated by a space
