In order to perform BIDSification of your data, your raw, unsorted data folder must contain the following files:

  • DataFiles - file containing raw eye position data (in .asc, .csv, or .tsv)

  • EventsFiles - file containing the events recorded during an experiment (in .csv, or .tsv)

  • infoFiles.tsv - file containing information about the data files in the directory

  • dataset_description.json - file describing the dataset

  • settings.json - file containing the general settings for the task

  • settingsEvents.json - file containing the description of the events recorded.

you can then BIDify your data (see Data standardisation)
