EventsFiles¶

Files containing the events recorded during an experiment (in .csv, or .tsv)

Warning

The data must be separated by a space
For exemple :¶
trial num_trial num_block dir_target proba switch
1 1 1 0.0 0.6475235198239354 0.0
2 2 1 1.0 0.6475235198239354 0.0
3 3 1 1.0 0.6475235198239354 0.0
